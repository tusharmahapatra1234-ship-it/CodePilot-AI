# Tech Stack

Project Name: CodePilot AI

# 1. Overview

CodePilot AI is built using a modern yet beginner-friendly technology stack. The selected technologies prioritize simplicity, scalability, maintainability, and future expansion.

---

# 2. Frontend

Technology:
HTML5

Purpose:
Defines the structure of the web application.

Why HTML?
- Lightweight
- Easy to maintain
- Browser compatible
- Semantic elements
- SEO friendly

Alternative:
React
Vue
Angular

Current Choice:
HTML5

Reason:
No build tools required and easier for Version 1.

---

Technology:

Purpose:
Application styling.

Features Used:
- Flexbox
- CSS Grid
- CSS Variables
- Animations
- Media Queries

Alternative:
Tailwind CSS
Bootstrap

Current Choice:
Pure CSS

Reason:
Complete design flexibility and no external dependency.

---

Technology:
JavaScript (ES6)

Purpose:
Frontend logic and API communication.

Features:
- Fetch API
- Async/Await
- DOM Manipulation

- Modules

Current Choice:
JavaScript

Reason:
Simple and sufficient for Version 1.
