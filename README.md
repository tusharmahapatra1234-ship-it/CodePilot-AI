# 📖 Overview

CodePilot AI is a web-based application that provides AI-assisted code analysis.

Users can paste source code, choose a programming language, select an AI action, and receive intelligent feedback generated by Google Gemini AI.

The project is designed to be modular, scalable, and beginner-friendly while following industry-standard software engineering practices.
